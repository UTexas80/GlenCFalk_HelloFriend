This is [Glen C. Falk's](https://glenfalk.rbind.io/) personal website. It is created with Hugo and blogdown, with the theme of *[hello-friend-ng](https://themes.gohugo.io/hugo-theme-hello-friend-ng/).* The source file of the website is hosted on Github and is served on netlify.
