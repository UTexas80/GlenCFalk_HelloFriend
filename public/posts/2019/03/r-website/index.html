Glen C. Falk’s Personal Website

This is Glen C. Falk’s personal website. It is created with Hugo and blogdown, with the theme of hello-friend-ng. The source file of the website is hosted on Github and is served on netlify.

Built With

  • blogdown- R package.
  • Netlify - Static web host.
  • rbind.io - Community blogs and websites for #rstats.
